HTML

HTML(HyperText Markup Language)은 웹 페이지를 만들 때 사용하는 마크업 언어야. 웹 문서의 구조를 정의하고, 텍스트, 이미지, 링크, 동영상 같은 요소를 배치하는 역할을 해.

CSS

CSS는 HTML 요소들의 디자인과 레이아웃을 정의하는 스타일 언어야. HTML이 웹 페이지의 **구조(뼈대)**를 담당한다면, CSS는 **디자인(색상, 크기, 배치 등)**을 담당해!

JAVASCRIPT

JavaScript는 웹 페이지를 동적으로 만들기 위해 사용하는 프로그래밍 언어야! HTML이 뼈대(구조), CSS가 **디자인(스타일)**을 담당한다면, **JavaScript는 동작(기능)**을 추가하는 역할을 해.
